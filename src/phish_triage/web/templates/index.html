What you'll see

A verdict card (benign / suspicious / malicious), the parsed authentication results, every suspicious signal as a pill, an IOC table (defanged URLs, hashed attachments, public Received-chain IPs), the scoring rationale, and the raw markdown report the CLI would print to stdout.
